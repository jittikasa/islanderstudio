# SEO Article Workflow Guide

This guide explains how to create and optimize articles for search engines using the Islander Studio Blog's Sanity CMS.

## Overview

The blog now includes comprehensive SEO features to help your content rank better in search engines and look great when shared on social media.

## Getting Started

### 1. Access the Sanity Studio

Navigate to your Sanity Studio at: `https://islanderstudio.sanity.studio`

### 2. Create a New Blog Post

1. Click on **"Blog Posts"** in the left sidebar
2. Click the **"Create"** button
3. Select **"Blog Post"**

## Content Fields

### Required Fields

#### **Title** ⭐
- Your main article title
- Recommended: 30-60 characters for optimal SEO
- This appears as the main heading on your blog
- If no custom SEO title is set, this will be used for search results

#### **Slug** ⭐
- Auto-generated from your title
- Creates the URL: `islanderstudio.com/blog/your-slug-here`
- Keep it short, descriptive, and keyword-rich
- Use hyphens to separate words

#### **Published At** ⭐
- When the article should be published
- Set to a future date to schedule publishing

#### **Excerpt** ⭐
- Short summary of your article (2-3 sentences)
- Recommended: 120-160 characters
- Used in blog listings and as fallback meta description
- Make it compelling to encourage clicks

#### **Body** ⭐
- Your main article content
- Supports rich text formatting:
  - Headings (H1-H4)
  - Bold and italic text
  - Code blocks
  - Blockquotes
  - Images with alt text

### Optional Fields

#### **Author**
- Select the article author
- Displays author info at the end of the article

#### **Main Image**
- Featured image for your article
- Recommended: 1200x800px or larger
- **Always add alt text** for accessibility and SEO
- Used as fallback for social sharing if no custom OG image is set

#### **Categories**
- Helps organize content
- Select relevant categories
- Improves internal navigation and SEO

#### **Reading Time**
- Estimated reading time in minutes
- Can be calculated automatically or manually set
- Displays in the post metadata

#### **Last Updated**
- When the article was last significantly updated
- Shows "Updated [date]" in the post header
- Signals freshness to search engines

#### **Related Apps**
- Link articles to PolaMoment or Shellist
- Articles appear in the "Related Reading" section of app pages

#### **Content Status**
- Track your article's optimization progress:
  - **Draft**: Initial writing phase
  - **In Review**: Ready for review
  - **SEO Optimized**: All SEO fields completed
  - **Published**: Live on the website

## SEO Settings (Advanced)

Click on the **"SEO Settings"** section to customize how your article appears in search results and social media.

### Meta Title
- Custom title for search results (different from main title)
- Recommended: 50-60 characters
- Include your target keyword
- Make it compelling to improve click-through rate
- **Example**: "10 Photography Tips for Beginners | Islander Studio"

### Meta Description
- Custom description for search results
- Recommended: 120-160 characters
- Include your focus keyword
- Write a compelling pitch to encourage clicks
- If not set, the excerpt will be used

### Focus Keyword
- Primary keyword you're targeting with this article
- **Example**: "photography tips" or "habit tracking"
- Use in your title, meta description, and throughout content naturally
- Aim for 0.5-2.5% keyword density

### Additional Keywords
- Other relevant keywords or phrases
- Add multiple keywords as tags
- Helps with semantic SEO
- **Example**: "beginner photography", "camera settings", "photo composition"

### Social Share Image (OG Image)
- Custom image for social media sharing
- Recommended: 1200x630px
- Should include text overlay or branding
- If not set, the main image will be used
- Always add alt text

### Canonical URL
- Override the canonical URL
- Use if the content was originally published elsewhere
- Prevents duplicate content issues
- Leave empty if this is the original source

### Hide from Search Engines
- Check this to prevent indexing (noindex)
- Useful for:
  - Draft content accidentally published
  - Temporary pages
  - Internal-only content

### Open Graph Type
- How social media platforms categorize your content
- Default: "Article" (recommended for blog posts)
- Options: Article, Website, Blog

## SEO Preview Panel

After filling in your content, click the **"SEO Preview"** tab to see:

### 1. SEO Score
- Overall optimization score (0-100%)
- **80%+**: Excellent
- **60-79%**: Good
- **Below 60%**: Needs improvement

### 2. Google Search Preview
- See exactly how your article will appear in Google search results
- Shows title, URL, and description
- Highlights if text is too long (truncated)

### 3. Character Counts
- Real-time tracking of title and description length
- Green = optimal length
- Orange = too long or too short

### 4. SEO Checklist
- Visual checklist of all SEO requirements
- Shows what's completed and what's missing
- Includes:
  - ✓ Has title
  - ✓ Title is 30-60 characters
  - ✓ Has excerpt
  - ✓ Has meta description
  - ✓ Has focus keyword
  - ✓ Has featured image
  - ✓ Featured image has alt text
  - ✓ Has categories

### 5. Recommendations
- Specific suggestions to improve SEO
- Prioritized list of what to fix first
- Updates in real-time as you edit

## Best Practices

### Content Writing

1. **Start with keyword research**
   - Identify what your audience is searching for
   - Use tools like Google Keyword Planner, Ahrefs, or SEMrush

2. **Structure your content**
   - Use headings (H2, H3) to organize content
   - Break up text into short paragraphs
   - Include bullet points and lists
   - Add relevant images with alt text

3. **Optimize for readability**
   - Write in active voice
   - Use simple language
   - Keep sentences short and clear
   - Aim for 8th-grade reading level

4. **Include internal links**
   - Link to other blog posts
   - Link to app pages (PolaMoment, Shellist)
   - Use descriptive anchor text

### SEO Optimization

1. **Title optimization**
   - Put focus keyword near the beginning
   - Make it compelling and clickable
   - Keep it under 60 characters

2. **Meta description optimization**
   - Include focus keyword naturally
   - Write a compelling call-to-action
   - Keep it 120-160 characters

3. **Image optimization**
   - Always add descriptive alt text
   - Include keywords naturally
   - Use descriptive file names
   - Compress images before uploading

4. **Keyword usage**
   - Use focus keyword in:
     - Title
     - First paragraph
     - At least one heading
     - Meta description
     - Image alt text
     - URL slug
   - Don't over-optimize (keyword stuffing)

### Workflow Process

#### Phase 1: Draft (Content Status: Draft)
1. Write your article content
2. Add main image
3. Write excerpt
4. Select categories
5. Set publish date

#### Phase 2: Review (Content Status: In Review)
1. Proofread content
2. Check formatting and structure
3. Verify all images have alt text
4. Review reading flow

#### Phase 3: SEO Optimization (Content Status: SEO Optimized)
1. Open SEO Settings
2. Set meta title
3. Write meta description
4. Add focus keyword
5. Add additional keywords
6. Create custom OG image (optional)
7. Check SEO Preview panel
8. Aim for 80%+ SEO score
9. Address all recommendations

#### Phase 4: Publish (Content Status: Published)
1. Final review of SEO preview
2. Verify publish date
3. Save and publish
4. Share on social media
5. Monitor performance

## Updating Existing Articles

When updating an article:

1. Make your content changes
2. Update the **"Last Updated"** field to the current date
3. Review and update SEO settings if needed
4. Check if meta description still accurately describes the content
5. Update the **Content Status** if moving through workflow stages

## Monitoring Performance

After publishing:

1. Monitor search rankings for target keywords
2. Track organic traffic in Google Analytics
3. Check click-through rates in Google Search Console
4. Update content based on performance data
5. Refresh old content quarterly

## Common Issues and Solutions

### Issue: SEO Score Below 80%
**Solution**: Check the recommendations panel and complete all required fields

### Issue: Title Too Long
**Solution**: Shorten to under 60 characters while keeping it descriptive

### Issue: No Social Share Image
**Solution**: Create a 1200x630px image with text overlay or use featured image

### Issue: Low Keyword Density
**Solution**: Naturally incorporate focus keyword in headings and content

### Issue: Missing Alt Text
**Solution**: Add descriptive alt text to all images
